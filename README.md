# telecomadmin-superman

获取电信光猫超级密码与配置文件，方便改桥接模式

如果你觉得有用的话，不要吝惜你的star，右上角点star是免费的哦~

## 使用方法

直接运行即可，如果光猫地址不是`192.168.1.1`，比如是`192.168.10.1`，则添加启动参数`-host 192.168.10.1`
